# 🚀 Personal Portfolio Website - ApexPlanet Task 4

## 📌 Project Overview

This project was developed as part of **Task 4 - Full Project Implementation** of the ApexPlanet Web Development Internship Program.

The project combines:

- Personal Portfolio Website
- To-Do Application with Local Storage
- Product Listing Page with Filtering and Sorting

---

## ✨ Features

### 👨‍💻 Personal Portfolio

- Professional Landing Page
- About Me Section
- Technical Skills Section
- Projects Showcase
- Contact Information

### 📋 To-Do Application

- Add Tasks
- Mark Tasks as Complete
- Delete Tasks
- Data Persistence using Local Storage

### 🛒 Product Listing Page

- Product Cards
- Category Filtering
- Price Sorting
- Product Ratings

### 📱 Responsive Design

- Mobile Friendly
- Tablet Friendly
- Desktop Friendly
